一、LAMP 简介

LAMP 指一组用于运行动态网站或服务器的自由软件名称首字母缩写:

  • Linux，操作系统，这里使用 Ubuntu
  • Apache，网页服务器，这里使用 Apache2
  • MariaDB 或 MySQL，数据库管理系统，这里使用 MySQL
  • PHP、Perl或Python，脚本语言，这里使用 PHP
    更多内容可以查阅维基百科了解 维基百科 LAMP
